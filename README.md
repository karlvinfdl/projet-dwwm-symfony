# BarberShop Booking – Projet DWWM

Application web de réservation pour un barbershop / salon de coiffure développée avec Symfony.

## Objectif

Ce projet permet aux clients de consulter les prestations du salon et de réserver un rendez-vous en ligne.
Un espace administrateur sécurisé permet de gérer les prestations, les utilisateurs et les réservations.

## Technologies utilisées

- PHP
- Symfony
- MySQL
- Doctrine
- Twig
- HTML / CSS / JavaScript

## Fonctionnalités principales

- Inscription et connexion des utilisateurs
- Consultation des prestations
- Réservation d’un créneau
- Gestion des réservations
- Interface administrateur
